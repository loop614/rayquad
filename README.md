## Description
- Check if the Vector2 is in the [quadrilateral](https://en.wikipedia.org/wiki/Quadrilateral)
- Move the Vector2 by left click
- Click button next for the next quadrilateral example
- Using raylib

## Quick Start
- update raylib paths in Makefile
```console
- make demo
```
